# 🍰 CakeShop - API
🚧 Em desenvolvimento

API REST para gerenciamento de produtos, pedidos e usuários (admin/clientes).

---

## 🚀 Tecnologias
- Java 21
- Spring Boot
- Spring Data JPA
- Spring Security
- PostgreSQL

---

## 📌 Funcionalidades
- Cadastro de usuários (com hash de senha)
- Login
- CRUD de produtos

---

## ▶️ Como rodar
```bash
mvn spring-boot:run
